## 경희대 축제 vlog

| 表达 | 意思 | 用法说明 |
|------|------|----------|
| 연습을 실전처럼 | 练习要当实战一样 | 처럼 = 像……一样 |
| 심장이 찢겨 나갈 거 같아 | 感觉心脏要被撕出来了 | 夸张表达紧张 |
| 내 나이가 어때서 | 我的年龄怎么了 | 来自经典歌曲 |
| 장난 아닌데 | 不是闹着玩的/真厉害 | 表示震惊的口语 |
| 기분이 좋아졌네 | 心情变好了呢 | -아/어지다 + -네 |
| 얼굴을 굽다 | 烤脸（指烤妆） | 视频核心玩笑，把上腮红比作"烤" |
| 광이 살아나다 | 光泽显现出来 | 化妆常用，살아나다=活过来 |
| 공들이는 만큼 | 花多少功夫就…… | -는 만큼=……的程度 |
| 시간 단축하기 좋다 | 很省时间 | 단축（缩短）是汉字词 |
| 마음에 들어요 | 很喜欢/满意 | 固定表达，直译"进到心里" |
| 기분 전환된다 | 心情转换了 | 전환（转换）是汉字词 |
| 꿀팁 | 超实用小窍门 | 꿀（蜂蜜）+ tip，网络流行语 |
| 화장이 먹다 | 妆服帖/吃妆 | 먹다 在这里=被吸收、贴合 |
| 공들이다 | 下功夫 | 공（功）+ 들이다（投入） |
| 영광의 상처 | 光荣的伤痕 | 영광 + 상처，都是汉字词 |

## 26학번 지원이요 15화

> 来源：아무따밴드 현장 & 차태현 아이돌 체험

| 表达 | 意思 | 用法说明 |
|------|------|----------|
| 큰일 났어 | 糟了/出大事了 | 차태현体验偶像时的口头禅 |
| 완전 계탔네 | 中大奖了/赚到了 | 意外占了大便宜 |
| 치고 빠지는 | 点到为止、见好就收 | 形容表演策略 |
| 접점을 찾다 | 找交集/找共同点 | "어떻게든 접점을 찾는 거예요" |

| 무대를 뒤집어 놓다 | 把全场点燃/掀翻舞台 | -아/어 놓다 的活用 |

## ENA《클라이맥스》제작발표회

| 표현 | 意思 | 用法说明 |
|------|------|----------|
| 부내가 느껴지다 | 能感觉到那股有钱劲儿 | 看到豪华/财阀气场时说 |
| 포스 뿜뿜(이다) | 气场/气势满满地往外冒 | 뿜뿜 = 拟态，"噗噗往外喷" |
| 가려운 곳이 긁히다 | 痒处被挠到 = 正合心意、解气 | 주지훈：시원하게 표현해줘서 가려운 곳이 긁히는 느낌 |
| 베일에 싸이다 | 笼罩在面纱中 = 神秘莫测 | 베일에 싸여진 인물 |
| 피 땀 눈물을 흘리다 / 영혼을 갈아서 만들다 | 流血流汗流泪 / 把灵魂磨进去 | 形容拼尽全力做的作品 |
| 열과 성을 다하다 | 倾注热情与诚意 | 正式·郑重的"全力以赴" |
| 힘에 부치다 | 力不从心、吃不消 | 각본 쓰는 작업이 힘에 부쳤다 |
| 발목이 잡히다 | 脚踝被抓住 = 被绊住/拖累 | 시청률에 발목이 잡히다 |
| 끈도 줄도 (인맥도) 없다 | 没门路没关系没人脉 | 三个近义词排比，强调"上面没人" |
| 머리가 하얘지다 | 脑子一片空白 | 紧张/被打乱时 |
| 토할 것 같다 | 紧张到想吐 | 夸张表达极度紧张 |
| 말 나온 김에 | 既然提到了、顺这话头 | 顺势展开话题的衔接语 |
| 대결 아닌 대결 | "非对决的对决" | A 아닌 A 句式：表面…实则未必… |
| 엔딩 맛집 | "结局名店" | 맛집=美食店，引申"某方面很拿手的（作品/人）" |
